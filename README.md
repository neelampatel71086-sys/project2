# project2# Python Menu-Driven Program

This is a simple Python menu-driven program that allows users to:

1. Generate a star (`*`) pattern
2. Analyze numbers as even or odd
3. Calculate the total sum of numbers in a range
4. Exit the program

---

# Features

## 1. Pattern Generator
- Accepts the number of rows from the user
- Prints a right-angled triangle star pattern
